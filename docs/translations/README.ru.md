## 📖 О KanaDojo

KanaDojo — это увлекательная веб-платформа для изучения японского языка, которая делает освоение хираганы, катаканы, кандзи и словарного запаса увлекательным и интуитивно понятным. KanaDojo, созданная с акцентом на эстетику, персонализацию и эффективность обучения, предлагает интерактивную образовательную среду для изучающих японский язык любого уровня.

Независимо от того, начинаете ли вы с базовой слоговой азбуки каны или готовитесь к экзаменам JLPT с углубленным изучением кандзи и лексики, KanaDojo предлагает оптимизированный процесс обучения без отвлекающих факторов, который адаптируется к вашим предпочтениям и стилю обучения.

### ✨ Ключевые возможности

#### 🎯 **Три тренировочных додзё**

- **Кана Додзё** - Освойте слоговые азбуки хираганы и катаканы с базовыми, дакуон, юн и иностранными звуковыми группами
- **Кандзи Додзё** - Изучите основные символы кандзи, организованные по уровням JLPT (N5, N4, N3, N2)
- **Словарь Додзё** - Расширяйте свой словарный запас японского языка с помощью подборок слов, подобранных по уровню владения языком

#### 🎮 **Четыре динамических игровых режима**

Каждое додзё поддерживает четыре увлекательных режима тренировок для закрепления знаний:

1. **Выбор** - Множественный выбор: выберите правильную романизацию/перевод для показанного символа.
2. **Обратный выбор** - Обратный множественный выбор: выберите правильный символ для данной романизации/перевода.
3. **Ввод** — Ввод текста: введите правильную латинизацию/перевод.
4. **Обратный ввод** — Обратный ввод текста: введите правильный символ.

#### 🎨 **Расширенные возможности кастомизации**

- **100+ тем** — Выберите из обширной коллекции красивых светлых и тёмных тем или используйте функцию случайной темы
- **28 японских шрифтов** — Выбирайте аутентичные японские шрифты в соответствии с вашими эстетическими предпочтениями
- **Звуковые эффекты** — Приятные звуковые отклики интерфейса, которые можно включать и отключать
- **Параметры отображения** — Переключение между ромадзи/английским и кана/кандзи в меню выбора
- **Горячие клавиши** — Комбинации клавиш для более эффективной тренировки (можно отключить)

#### 📊 **Отслеживание прогресса**

- Обратная связь в режиме реального времени с указанием правильных/неправильных результатов
- Отслеживание серий для поддержания мотивации
- Статистика для отслеживания прогресса в обучении

#### 🌐 Современный веб‑опыт

- Полностью адаптивный дизайн — работает на компьютерах, планшетах и мобильных устройствах
- Не требует установки — тренируйтесь где угодно при наличии интернет‑соединения
- Чистый и минималистичный интерфейс, позволяющий сосредоточиться на учёбе
- Плавные анимации и переходы, реализованные с помощью Framer Motion

<a id="ui-design-philosophy"></a>
## 🎨 UI и философия дизайна

KanaDojo сочетает в себе **минималистичную эстетику** и **максимальную гибкость**. Философия дизайна основана на:

### Минимализм прежде всего

- Чистые интерфейсы с минимумом отвлекающих факторов
- Фокус на учебном материале
- Интуитивно понятная навигация и чёткая иерархия информации
- Целенаправленное использование свободного пространства

### Эстетическая кастомизация 

- Обширная библиотека тем (более 100 вариантов) – от нежных пастельных до ярких неоновых
- Поддержка светлого и тёмного режимов
- Тщательно подобранные цветовые палитры, приятные для глаз во время длительных занятий
- Плавные переходы тем

### Пользовательский опыт

- Плавная анимация и микровзаимодействия для приятной обратной связи
- Адаптивный дизайн, который прекрасно адаптируется к любому размеру экрана
- Звуковая обратная связь для взаимодействия (опционально)
- Единый визуальный язык во всех разделах

### Японская типография

- 28 аутентичных японских шрифтов, охватывающих различные стили
- Корректная визуализация сложных иероглифов
- Чёткое различие между похожими символами
- Предварительный просмотр шрифтов с реальными образцами японского текста

## 🛠️ Технический стек

KanaDojo создан с использованием современных веб-технологий для оптимальной производительности и удобства разработчиков:

- **[TypeScript](https://www.typescriptlang.org/)** - Типобезопасная разработка

### Стиль и пользовательский интерфейс

- **[Tailwind CSS](https://tailwindcss.com/)** - CSS-фреймворк, ориентированный на утилиты
- **[shadcn/ui](https://ui.shadcn.com/)** - Высококачественная и доступная библиотека компонентов

- **Zustand Persist** — Сохранение настроек пользователя в локальное хранилище

### Утилиты и функции

<a id="getting-started"></a>
## 🚀 Начало работы

### Предварительные условия

- **Node.js** 18.x или новее
- **npm** 10.x или новее (поставляется вместе с Node.js)

### Установка

1. **Клонируйте репозиторий**

   ```bash
   git clone https://github.com/lingdojo/kanadojo.git
   cd kanadojo
   ```

2. **Установите зависимости**

   ```bash
   npm install
   ```

3. **Запустите сервер разработки**

   ```bash
   npm run dev
   ```

4. **Откройте браузер**

   Перейдите по адресу: [http://localhost:3000](http://localhost:3000)

### Сборка для производства

```bash
# Создать оптимизированную производственную сборку
npm run build

# Запустить production‑сервер
npm start
```

### Другие команды

```bash
# Запустить ESLint
npm run lint

# Сгенерировать карту сайта (выполняется автоматически после сборки)
npm run postbuild
```
### Поиск неисправностей

Если во время разработки возникли проблемы, попробуйте следующие решения:

#### Очистка кэша Next.js

**macOS/Linux:**
```bash
rm -rf .next
npm run dev
```

**Windows (PowerShell):**
```powershell
Remove-Item -Recurse -Force .next
npm run dev
```

**Windows (Command Prompt):**
```cmd
rmdir /s /q .next
npm run dev
```

#### Удаление node_modules и повторная установка

**macOS/Linux:**
```bash
rm -rf node_modules package-lock.json
npm install
```

**Windows (PowerShell):**
```powershell
Remove-Item -Recurse -Force node_modules, package-lock.json
npm install
```

**Windows (Command Prompt):**
```cmd
rmdir /s /q node_modules
del package-lock.json
npm install
```

#### Полная очистка кэшей (Nuclear option)

**macOS/Linux:**
```bash
rm -rf .next node_modules package-lock.json
npm cache clean --force
npm install
npm run dev
```

**Windows (PowerShell):**
```powershell
Remove-Item -Recurse -Force .next, node_modules, package-lock.json
npm cache clean --force
npm install
npm run dev
```

**Windows (Command Prompt):**
```cmd
rmdir /s /q .next
rmdir /s /q node_modules
del package-lock.json
npm cache clean --force
npm install
npm run dev
```

#### Порт уже используется

Если порт 3000 уже используется:

**macOS/Linux:**
```bash
# Найти процесс, использующий порт 3000
lsof -i :3000

# Завершить процесс (замените PID на реальный идентификатор процесса)
kill -9 PID
```

**Windows (PowerShell/Command Prompt):**
```cmd
# Найти процесс, использующий порт 3000
netstat -ano | findstr :3000

# Завершить процесс (замените PID на реальный идентификатор процесса)
taskkill /PID PID /F
```

Или запустите сервер на другом порту:

```bash
# macOS/Linux/Windows
PORT=3001 npm run dev
```

### Ключевые концепции

---

#### Поток управления состоянием

1. Пользователь выбирает контент через элементы меню
2. Выборы сохраняются в сторах Zustand (`useKanaKanjiStore`, `useVocabStore`)
3. Компоненты тренировки читают данные из сторов и генерируют вопросы
4. Статистика отслеживается и сохраняется в `useStatsStore`
5. Предпочтения пользователя сохраняются в `useThemeStore` с персистентностью в localStorage

#### Архитектура компонентов

- **Компоненты додзё**: Отвечают за выбор символов/слов для каждого типа контента
- **Компоненты тренировки**: Отображают игровые режимы и обрабатывают взаимодействие с пользователем
- **Переиспользуемые компоненты**: Общие элементы интерфейса (кнопки, карточки, модальные окна и т.д.)
- **Компоненты меню**: Навигация, информационные секции и выбор додзё

#### Организация данных

- **Кана**: Организовано по типу (хирагана/катакана) и по группам (базовая, dakuon, yoon, иностранные)
- **Кандзи**: Организовано по уровням JLPT (N5–N2) с чтениями и значениями
- **Словарь**: Организовано по уровням JLPT и типу слова (существительные, глаголы и т.д.)

#### Реализация игровых режимов

Каждый игровой режим реализован как динамический маршрут (`/[contentType]/train/[gameMode]`), который:

1. Считывает выбранный контент из соответствующего стора
2. Генерирует случайные вопросы из выбранного набора
3. Даёт мгновенную обратную связь
4. Отслеживает статистику (правильные, неправильные ответы, серия)
